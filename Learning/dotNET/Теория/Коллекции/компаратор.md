**Компаратор** в C# — это объект, который реализует интерфейс `IComparer<T>` и задаёт пользовательскую логику для сравнения объектов. Он используется для сортировки коллекций (например, списков) с кастомным правилом.

### **Как задать компаратор для списка:**

1. **Создание класса компаратора:**  
    Реализуйте интерфейс `IComparer<T>` с методом `Compare`.

```csharp
public class CustomComparer : IComparer<int>
{
    public int Compare(int x, int y)
    {
        // Сравниваем числа в обратном порядке
        return y.CompareTo(x);
    }
}
```

2. **Использование компаратора:**  
    Передайте экземпляр компаратора в метод `Sort`.

```csharp
var numbers = new List<int> { 5, 2, 8, 3 };
numbers.Sort(new CustomComparer()); // Сортировка в убывающем порядке
Console.WriteLine(string.Join(", ", numbers)); // 8, 5, 3, 2
```

---

### **Альтернативный способ: Лямбда-выражение**

Для одноразовых правил сортировки можно использовать перегрузку метода `Sort` с делегатом.

```csharp
var numbers = new List<int> { 5, 2, 8, 3 };
numbers.Sort((x, y) => y.CompareTo(x)); // Сортировка в убывающем порядке
Console.WriteLine(string.Join(", ", numbers)); // 8, 5, 3, 2
```

### **Когда использовать компаратор?**

- Для сложных или часто используемых правил сортировки (лучше отдельный класс).
- Для одноразовых случаев (удобнее лямбда).
