**Динамическая коллекция, которая может изменять размер во время выполнения.**

**Особенности:**
- Простота добавления, удаления и поиска элементов.
- Используется класс `List<T>`.
- не потокобезопасный

```C#
List<string> people = new List<string>();
List<string> people = new List<string>() { "Tom", "Bob", "Sam" };
var people = new List<string>() { "Tom", "Bob", "Sam" };
var employees = new List<string>(people);
var employees = new List<string>(people){"Mike"};
List<string> people = ["Tom", "Bob", "Sam"];
List<string> employees = [];// пустой список

// Установка начальной емкости списка
List<string> people = new List<string>(16);
```

**Длина списка**
```C#
var people = new List<string>() { "Tom", "Bob", "Sam" };
Console.WriteLine(people.Count);    // 3
```

**Перебор списка**
```C#
var people = new List<string>() { "Tom", "Bob", "Sam" };
 
foreach (var person in people)
{
    Console.WriteLine(person);
}
for (int i = 0; i < people.Count; i++)
{
    Console.WriteLine(people[i]);
}
```




- **Вопрос:** Какой алгоритм используется для динамического увеличения `List`?  
    **Ответ:** При достижении текущего размера, размер внутреннего массива увеличивается в 2 раза.
    
- **Вопрос:** Как узнать текущую емкость списка?  
    **Ответ:** Используйте свойство `Capacity`.
    
- **Вопрос:** Чем отличаются методы `Add` и `Insert`?  
    **Ответ:** `Add` добавляет элемент в конец списка, `Insert` вставляет элемент по указанному индексу.
    
- **Вопрос:** Как удалить элемент из списка?  
    **Ответ:**    
    - По значению: `list.Remove(value)`.
    - По индексу: `list.RemoveAt(index)`.
- **Вопрос:** Что такое `Find` в `List`?  
    **Ответ:** Находит первый элемент, удовлетворяющий условию:
    `int result = list.Find(x => x > 10);`
    
- **Вопрос:** Чем отличается `Contains` от `Exists`?  
    **Ответ:**
    - `Contains` проверяет наличие элемента.
    - `Exists` принимает лямбда-выражение для проверки условия.
- **Вопрос:** Как отсортировать `List`?  
    **Ответ:** Используйте `list.Sort()` или передайте [[компаратор]].
    
- **Вопрос:** Какой используются алгоритм сортировки?  
    **Ответ:** Начиная с .NET Framework 4.5, метод `List<T>.Sort()` использует алгоритм `TimSort`, TimSort был разработан Тимом Петерсом (Tim Peters) для Python в 2002 году и используется в стандартной библиотеке Python..
    
- **Вопрос:** Как сделать `List` неизменяемым?  
    **Ответ:** Используйте `AsReadOnly`.
    
- **Вопрос:** Как работает метод `BinarySearch`?  
    **Ответ:** Выполняет двоичный поиск, но список должен быть предварительно отсортирован.
    
- **Вопрос:** Как получить подсписок?  
    **Ответ:** Используйте `list.GetRange(start, count)`.
