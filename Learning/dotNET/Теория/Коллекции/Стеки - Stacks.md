**Структура данных "последним пришел — первым вышел" (LIFO).**

**Особенности:**
- Элементы добавляются и извлекаются с одного конца.
- Используется класс `Stack<T>`.
```C#
Stack<string> people = new Stack<string>();
Stack<string> people = new Stack<string>(16); // указание емкости

// инициализировать очередь элементами из другой коллекции или массивом
var employees = new List<string> { "Tom", "Sam", "Bob" };
Stack<string> people = new Stack<string>(employees);
foreach (var person in people) Console.WriteLine(person);
```

Для перебора очереди можно использовать стандартный цикл **foreach**.

Для получения количества элементов в очереди в классе определено свойство **Count**.


- **Вопрос:** Как работает метод `Push`?  
    **Ответ:** Добавляет элемент на вершину стека.
    
- **Вопрос:** Чем `Pop` отличается от `Peek`?  
    **Ответ:**    
    - `Pop` удаляет и возвращает элемент.
    - `Peek` возвращает элемент без удаления.
- **Вопрос:** Как проверить, пуст ли стек?  
    **Ответ:** Используйте метод `Count == 0`.
    
- **Вопрос:** Как работает стек в рекурсии?  
    **Ответ:** Вызовы методов и локальные переменные хранятся в стеке.
    
- **Вопрос:** Как использовать стек для проверки скобок?  
    **Ответ:** Добавляйте открывающие скобки, удаляйте при встрече закрывающих.
    
- **Вопрос:** Как итерировать стек?  
    **Ответ:** С помощью `foreach`, начиная с вершины.
    
- **Вопрос:** Что произойдет при вызове `Pop` для пустого стека?  
    **Ответ:** Исключение `InvalidOperationException`.
    
- **Вопрос:** Чем стек отличается от очереди?  
    **Ответ:** Стек — LIFO, очередь — FIFO.
    
- **Вопрос:** Можно ли стек сделать потокобезопасным?  
    **Ответ:** Используйте `ConcurrentStack`.
    
- **Вопрос:** Как перевернуть строку с помощью стека? 
    **Ответ:** Поместите символы строки в стек, а затем извлеките их обратно.
