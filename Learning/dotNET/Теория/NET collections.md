|---|---|---|---|---|---|
|**Array**|O(n)|Невозможно (фикс. размер)|Невозможно (фикс. размер)|O(1)|Быстрый доступ по индексу, фиксированный размер.|
|**List<T>**|O(n)|O(1)* в конец, O(n)** в середину|O(n)|O(1)|Динамический массив, автоматически изменяет размер.|
|**LinkedList<T>**|O(n)|O(1) для начала/конца, O(n)** для середины|O(n)|O(n)|Быстрое добавление/удаление в начале и конце, но медленный доступ к элементам.|
|**HashSet<T>**|O(1)|O(1)|O(1)|Не поддерживается|Эффективная проверка на наличие элементов, неупорядоченная коллекция.|
|**SortedSet<T>**|O(log n)|O(log n)|O(log n)|Не поддерживается|Упорядоченная коллекция, элементы хранятся в отсортированном порядке.|
|**Dictionary<TKey, TValue>**|O(1)|O(1)|O(1)|Не поддерживается|Быстрый доступ по ключу, неупорядоченная коллекция.|
|**SortedDictionary<TKey, TValue>**|O(log n)|O(log n)|O(log n)|Не поддерживается|Упорядоченная коллекция, доступ по ключу.|
|**Queue<T>**|O(n)|O(1)|O(1)** (dequeue)|Не поддерживается|Очередь (FIFO), быстрые операции на концах.|
|**Stack<T>**|O(n)|O(1)|O(1)** (pop)|Не поддерживается|Стек (LIFO), быстрые операции на концах.|
|**ConcurrentDictionary<TKey, TValue>**|O(1)|O(1)|O(1)|Не поддерживается|Потокобезопасный словарь, оптимизированный для многопоточной среды.|
|**ConcurrentQueue<T>**|O(n)|O(1)|O(1)** (dequeue)|Не поддерживается|Потокобезопасная очередь, оптимизированная для многопоточной среды.|
|**ConcurrentStack<T>**|O(n)|O(1)|O(1)** (pop)|Не поддерживается|Потокобезопасный стек.|
|**ImmutableList<T>**|O(n)|O(n)|O(n)|O(1)|Неизменяемая коллекция, безопасна для многопоточности.|
|**ImmutableDictionary<TKey, TValue>**|O(log n)|O(log n)|O(log n)|Не поддерживается|Неизменяемый словарь, безопасен для многопоточности.|
