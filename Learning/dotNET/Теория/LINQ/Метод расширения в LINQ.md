Метод расширения в LINQ создается с использованием статического класса и статического метода, который расширяет функциональность существующих типов (например, коллекций) без необходимости изменять их код.

### **Шаги для создания метода расширения в LINQ:**

1. Создайте статический класс, который будет содержать метод расширения.
2. Создайте статический метод внутри этого класса. Он должен быть `public` и `static`, и первый параметр должен использовать ключевое слово `this`, чтобы указать тип, который расширяется.
3. Определите логику метода расширения.

### **Пример создания метода расширения для LINQ:**

Давайте создадим метод расширения `WhereEven`, который будет фильтровать только четные числа в коллекции `IEnumerable<int>`.

```csharp
using System;
using System.Collections.Generic;

public static class LinqExtensions
{
    // Метод расширения для фильтрации четных чисел
    public static IEnumerable<int> WhereEven(this IEnumerable<int> source)
    {
        foreach (var item in source)
        {
            if (item % 2 == 0)
                yield return item;
        }
    }
}

class Program
{
    static void Main()
    {
        var numbers = new List<int> { 1, 2, 3, 4, 5, 6 };

        // Используем метод расширения WhereEven
        var evenNumbers = numbers.WhereEven();

        foreach (var number in evenNumbers)
        {
            Console.WriteLine(number);  // Выведет 2, 4, 6
        }
    }
}
```

### **Объяснение:**

1. **Статический класс** `LinqExtensions`: Этот класс содержит статический метод, который расширяет функциональность `IEnumerable<int>`.
2. **Метод расширения** `WhereEven`: Первый параметр метода имеет ключевое слово `this`, что позволяет методу быть доступным как метод для типа `IEnumerable<int>`.
3. **Логика метода**: Метод перебирает элементы коллекции и возвращает только те, которые являются четными числами.

### **Как использовать метод расширения**:

После того как метод расширения определен, вы можете использовать его как обычный метод LINQ. В примере выше `numbers.WhereEven()` фильтрует четные числа из списка.

### **Важные моменты:**

- Метод расширения должен быть доступен через `using`, если он определен в другом пространстве имен.
- Методы расширения применимы только к тем типам, для которых они созданы (в нашем примере — к `IEnumerable<int>`).
