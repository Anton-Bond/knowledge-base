Механизм `init` в C# позволяет задать значение свойства только один раз — при инициализации объекта. После этого изменить значение нельзя. Это удобно для создания immutable-объектов.

**Пример использования:**

```csharp
public class Person
{
    public string Name { get; init; }
    public int Age { get; init; }
}

var person = new Person
{
    Name = "Alice",
    Age = 30
};

// person.Name = "Bob"; // Ошибка: свойство только для инициализации.
```

**Особенности:**

1. Значение можно задать только:
    - В инициализаторе объекта (`object initializer`).
    - В конструкторе.
2. После завершения инициализации объект становится неизменяемым.
3. Это улучшает безопасность данных и делает объекты thread-safe.
