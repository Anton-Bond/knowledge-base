**Композиция** в C# — это подход к построению объектов, когда один класс включает в себя другой класс как часть своей реализации, вместо наследования.

**Особенности композиции:**

1. **Гибкость:** Позволяет изменять поведение, заменяя включаемые объекты.
2. **Избежание проблем наследования:** Нет жесткой связи между классами, как при наследовании.
3. **Повторное использование кода:** Один объект может быть использован в разных классах.

**Пример композиции:**

```csharp
public class Engine
{
    public void Start() => Console.WriteLine("Engine started");
}

public class Car
{
    private Engine _engine = new Engine();
    
    public void Drive()
    {
        _engine.Start();
        Console.WriteLine("Car is driving");
    }
}

// Использование:
var car = new Car();
car.Drive();
```

**Когда использовать:**

- Когда отношения между классами описываются как "имеет" (`has-a`), например, машина имеет двигатель.
- Для увеличения модульности и гибкости кода.
